Gravatar Generator widget for XE

Created by Juwan Park
http://parkjuwan.dothome.co.kr/

그라바타를 xe에서 사용할 수 있게 하는 위젯입니다.

설치 방법
* xe > widgets 디렉터리 안에 gravagen 디렉터리를 만들어서 업로드합니다.
* Upload gravagen directory in xe > widgets dir.
